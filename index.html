Valorant is a game made by Riot, this game is a 5v5 free-to-play game, the game takes place in the future, each game character has their own unique abilities, and the core gameplay focuses on tactics and team-based shooting.

Valorant is a team-based first-person hero shooter set in the near future.Players play as one of a set of Agents, characters designed based on several countries and cultures around the world.
